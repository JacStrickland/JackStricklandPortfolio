Physical Computing / Jan 27, 2025

SafeKeeping Lock Box

A functional smart lockbox prototype that uses infrared remote input, microcontroller logic, servo-based locking, and LED feedback to create a clear access-control interaction.

ClientHCDE 439
RoleDesigner & Project Manager
FocusEmbedded interaction, hardware, feedback systems

Background

Defining an intuitive system

SafeKeeping was designed as an interactive lockbox that combines physical security with clear user feedback. The goal was to protect personal items while communicating system state through visible red and green LED indicators.

Crafting a reliable interaction

The prototype was refined through iterative testing of infrared input, code validation, locking behavior, and servo movement. The interaction balances technical reliability with usability by giving users immediate confirmation when the box is locked or unlocked.

Work highlight

The final system integrates infrared input, microcontroller logic, and servo actuation into a standalone embedded prototype. It demonstrates strengths in physical computing, system design, and translating technical behavior into a user-centered interaction.
